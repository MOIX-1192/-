# 炉石传说模拟抽卡数据包
## 使用指南
本仓库为炉石开包模拟插件的数据包，请使用前先下载好插件  
该插件需要mirai支持，请自行安装  
下载后数据包（主要数据在卡牌文件夹下）后置于~\Mirai\plugins\MiraiNative\data路径下   


![图片](https://img01.sogoucdn.com/app/a/100520146/4ddf5d3102f27c10b9a99923e2018c14)




然后打开控制台更新数据即可
